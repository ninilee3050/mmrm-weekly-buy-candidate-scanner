1. 종목: [티커 1] / 기준봉: [YYYY-MM-DD]
2. 종목: [티커 2] / 기준봉: [YYYY-MM-DD]
3. 종목: [티커 3] / 기준봉: [YYYY-MM-DD]

각 종목에 대해 다음을 비교해줘.
- 패턴명과 패턴 유사도 상태
- 8주 상승확률
- 52주 뒤 +20% 확률
- 52주 내 +20% 도달확률과 중앙 소요기간
- 52주 최대하락폭
- ATH 안전마진
- 150·200주선과 MACD
- 투자 논리가 성립하는 기대
- 조심할 점
- 매수 또는 매수하지 않음

기준봉 이후 실제 결과는 순위와 매수판정에 사용하지 마.
마지막에는 사용자의 `1년 +20% 목표`에 가장 적합한 순위를 표로 정리해줘.

============================================================
공통 분석 규칙
============================================================

- 공식 기준은 investing_proxy이고 KIS 호환값은 민감도 비교로만 사용한다.
- 과거 사건이면 data/events_investing_proxy.csv와 data/walk_forward_predictions.csv를 우선한다.
- 신규 사건이면 model/pattern_model.json과 tools/pattern_classifier.py로 패턴을 분류한다.
- 신규 사건의 확률은 data/pattern_metrics.csv의 같은 패턴 공식값을 사용한다.
- 데이터가 부족하면 수치를 꾸며내지 말고 `분석자료 부족`이라고 말한다.
- 패턴 확률은 과거 집단빈도이며 개별 종목의 실제 결과를 보장하지 않는다.
